# Pearl Path

Travel planner front-end concept for discovering destinations, organizing trips, and tracking budget in one clean flow.

![Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JavaScript-111827)
![Type](https://img.shields.io/badge/Project-Multi--Page%20UI-10B981)

## What It Shows

- Hero landing page with slideshow and product messaging
- Client-side auth simulation (login/signup redirect flow)
- Dashboard with journey cards and planning panel
- Explore page with search + category filtering
- Budget page with expense summary and feedback interaction

## Demo Flow

1. Open index.html
2. Click Get Started and sign in
3. Navigate through Dashboard, Explore, and Budget

## Main Pages

| Page | Purpose |
|---|---|
| index.html | Landing and first impression |
| login.html / signup.html | Demo auth flow |
| dashboard.html | Trip overview and planning UI |
| explore.html | Destination discovery experience |
| budget.html | Expense tracking interface |

## Run Locally

This is a static project, so no install/build step is required.

- Open the folder in VS Code
- Launch index.html in browser
- Optional: run with Live Server for easier navigation

## Notes

- Data is currently mock/static
- Auth state is stored in localStorage using isLoggedIn
- Protected routes: dashboard.html, explore.html, budget.html
